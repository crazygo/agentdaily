JetBrains Agent Client Protocol (ACP)

JetBrains announced the Agent Client Protocol (ACP), a standardized communication protocol between IDEs and coding agents, similar to how LSP works for language servers. Released in beta in version 25.3, ACP allows developers to use any ACP-compatible agent in JetBrains IDEs.

Karpathy's 'Vibe Coding' Paradigm Gains Traction

Andrej Karpathy's concept of 'vibe coding' - where developers fully embrace AI assistance and rely on intuition rather than traditional coding practices - has become a major discussion point. The approach involves coding through natural language conversation with AI models, focusing on problem-solving atmosphere rather than code details.
