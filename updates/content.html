Andrej Karpathy: Claude Coding and the AI Programming Era

Andrej Karpathy posts a viral tweet about his experience with Claude coding, noting that Claude and Codex crossed a critical 'coherence threshold' around December 2025. His workflow shifted from ~80% manual coding to ~80% AI agent-based since November 2025, with improvements in Claude Opus 4.5 as a key factor. Jan 27 Twitter

Multi-task Code LLMs: Data Mix or Model Merge?

Researchers from Rensselaer Polytechnic Institute, Cisco, and IBM publish a study comparing data mixing versus model merging strategies for creating multi-task code LLMs. The research finds that model merging achieves better performance at larger scale (7B parameters), retaining 96% of specialized model performance on code generation tasks.

Jan 28 ArXiv

Fingerprinting AI Coding Agents on GitHub

Researchers from Trent University present the first study on fingerprinting AI coding agents, analyzing 33,580 PRs from five major agents (OpenAI Codex, GitHub Copilot, Devin, Cursor, Claude Code). Using 41 features spanning commit messages, PR structure, and code characteristics, they achieve 97.2% F1-score in multi-class agent identification.

Jan 24 ArXiv

How AI Assistance Impacts the Formation of Coding Skills

Anthropic publishes research based on observational study of Claude.ai data, examining how AI helps people complete parts of their job faster. The study investigates the impact of AI assistance on skill development, productivity, and learning outcomes in software engineering contexts.
