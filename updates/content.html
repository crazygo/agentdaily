My LLM Coding Workflow Going Into 2026

Addy Osmani shares his comprehensive AI-assisted engineering workflow, emphasizing that ~90% of Claude Code's code is now written by Claude Code itself. Key principles include: 1) Start with clear specs before coding, 2) Break work into small iterative chunks, 3) Provide extensive context and guidance, 4) Choose the right model for each task, 5) Leverage AI across the software development lifecycle, 6) Keep human in the loop with verification and testing, 7) Commit often and use version control as safety net, 8) Customize AI behavior with rules and examples, 9) Embrace testing and automation, 10) Continuously learn and adapt. Osmani advocates for 'AI-augmented software engineering' rather than fully automated development.
