Introducing the Codex app

OpenAI released a new macOS app for their Codex coding agent. Key features: first-class support for Skills, Automations for running scheduled tasks (tracks state in SQLite), built with Electron for cross-platform support (Windows coming soon). Usage stats: since GPT-5.2-Codex launch in mid-December, overall Codex usage has doubled, and in the past month more than a million developers have used Codex. Codex is described as 'a general agent harness disguised as a tool for programmers'.
