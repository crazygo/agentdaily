Jan 27 Anthropic Research

Fingerprinting AI Coding Agents on GitHub

Researchers from Trent University present the first study on fingerprinting AI coding agents, analyzing 33,580 PRs from five major agents. Using 41 features spanning commit messages, PR structure, and code characteristics, they achieve 97.2% F1-score in multi-class agent identification.

Jan 24 ArXiv

How AI Coding Agents Modify Code: Large-Scale Study

Researchers publish a large-scale study on arXiv examining how AI coding agents modify code, analyzing GitHub pull requests involving AI coding agents. The study provides empirical evidence about the patterns, quality, and characteristics of AI-generated code changes in real-world projects.

Jan 24 ArXiv

The Math on AI Agents Doesn't Add Up

WIRED publishes an analysis by Steven Levy arguing that the economic and practical assumptions behind AI coding agents may be flawed. The piece examines mathematical limitations of current LLM architectures, citing research on hallucination rates and the fundamental probabilistic nature of language models.

FastRender: Browser Built by Thousands of Parallel Agents

Simon Willison interviews Wilson Lin from Cursor about FastRender, a web browser built from scratch using swarms of autonomous AI coding agents. The project used approximately 2,000 agents running concurrently, making nearly 30,000 commits and thousands of commits per hour.

Jan 23 Simon Willison

Multi-task Code LLMs: Data Mix or Model Merge?

Researchers from Rensselaer Polytechnic Institute, Cisco, and IBM publish a study comparing data mixing versus model merging strategies for creating multi-task code LLMs. The research finds that model merging achieves better performance at larger scale (7B parameters), retaining 96% of specialized model performance on code generation tasks.
